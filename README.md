A beginner-friendly demonstration project created to learn and showcase Git version control fundamentals.

## 🚀 Getting Started

### Prerequisites
- Git installed on your system
- A GitHub account

### Steps to Clone This Repository

```bash
# Clone the repository
git clone https://github.com/<your-username>/git-demo-project.git

# Navigate into the folder
cd git-demo-project
```

---

## 📖 What I Learned

- How to initialize a Git repository using `git init`
- Staging files with `git add`
- Committing changes with `git commit -m "message"`
- Connecting to a remote repository using `git remote add origin`
- Pushing code to GitHub using `git push`
- Writing clear documentation with Markdown
